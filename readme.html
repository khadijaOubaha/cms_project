# Boutique Artisanale - WordPress Ce projet est une boutique en ligne développée avec WordPress, dédiée à la vente de produits artisanaux, incluant des articles de décoration, des vêtements et des accessoires artisanaux uniques. ## **Fonctionnalités principales** - **Décor** : Vente d'articles de décoration artisanale tels que tapis, poufs et accessoires décoratifs uniques. - **Vêtements** : Propose des vestes brodées, babouches et autres vêtements artisanaux. - **Best Sellers** : Met en avant les produits les plus populaires auprès des clients. --- ## **Instructions d'installation** ### **1. Prérequis** - Serveur web local ou distant avec PHP 7.4 ou supérieur (XAMPP). - Base de données MySQL. - Accès au tableau de bord WordPress. ### **2. Étapes d'installation** 1. **Cloner le dépôt ou télécharger les fichiers du projet** : ```bash git clone https://github.com/khadijaOubaha/cms_project.git ``` Ou téléchargez les fichiers en tant qu'archive ZIP depuis GitHub. 2. **Déplacer les fichiers** : - Placez les fichiers dans le répertoire racine de votre serveur web (ex. `htdocs` pour XAMPP). 3. **Créer une base de données** : - Accédez à `phpMyAdmin` et créez une nouvelle base de données (`Artisanat_db`). 4. **Importer la base de données** : - Importez le fichier SQL inclus dans le dossier du projet (`artisanat_db.sql`). 5. **Configurer WordPress** : - Accédez à votre site via le navigateur (ex. `http://localhost/Artisanat`). - Suivez les étapes de configuration de WordPress. - Lorsqu'on vous demande les informations de la base de données, entrez les détails appropriés (nom de la base de données, utilisateur, mot de passe). 6. **Configurer le thème et les plugins** : - Activez le thème personnalisé disponible dans le projet (ayroma). - Installez et activez les plugins nécessaires (Gutenberg, WooCommerce,Elementor). --- ## **Identifiants administrateur principal** Pour accéder au tableau de bord WordPress en tant qu'administrateur principal : - **URL de connexion** : `http://localhost/Artisanat/wp-admin` - **identifiant ** : `khadija_safa` - **Mot de passe** : `r!Fp$zBuT5xGkt7zEU` > **Note** : Pensez à modifier ces identifiants après la première connexion pour des raisons de sécurité. --- ## **Aide et support** Si vous rencontrez des problèmes lors de l'installation ou de l'utilisation, n'hésitez pas à me contacter à l'adresse suivante : - **GitHub** : [khadijaOubaha](https://github.com/khadijaOubaha/) ou [safabelhoufrte](https://github.com/safabelhoufrte)
